## Codebase Patterns
- `.npmrc` at repo root overrides Amazon CodeArtifact with `registry=https://registry.npmjs.org/`
- Git identity is repo-scoped: `harryfgoatcher@gmail.com` (not global Amazon email)
- `DATABASE_URL` (port 6543) for app queries; `DIRECT_URL` (port 5432) for migrations/scripts
- shadcn/ui requires `class-variance-authority`, `clsx`, `tailwind-merge` as explicit deps
- T3 scaffold post table/router removed in US-002 — replaced with health router placeholder
- `vercel env add <key> preview "" --yes` — empty string 3rd arg skips interactive branch prompt
- `SKIP_ENV_VALIDATION=true` in CI to skip env validation during build
- Vitest config at `vitest.config.ts` with `~` alias pointing to `./src`
- Tests go in `tests/` directory (not `src/__tests__/`)
- vitest needs `setupFiles: ["dotenv/config"]` for DB-connected tests
- Tables use raw names (no `yieldlens_` prefix) — drizzle.config.ts has no tablesFilter
- Empty tRPC router breaks createHydrationHelpers type — always have ≥1 procedure
- Custom SQL (triggers, functions, matview) applied separately from Drizzle push — not managed by drizzle-kit
- Matview refresh deferred until after data ingestion (empty tables produce empty view)
