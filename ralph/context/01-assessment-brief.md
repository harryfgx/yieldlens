## Key Deliverables

### 1. Data Model
Must include:
- a) Minimum 5 entities
- b) Attributes and data types
- c) PKs and FKs
- d) Relationships with cardinalities
- e) Justifications of noteworthy design decisions
- f) 3NF normalisation explanation

### 2. Database Implementation
Must include:
- a) Full database schema as SQL queries
- b) Sample data — appropriate quantity and quality, real (OGL-compliant) or synthetic
- c) Implementation process explanation including tools used (GenAI for SQL is explicitly permitted)
- Evidence via screenshots, references, SQL as text

### 3. Data Reporting and Visualisation
Must include:
- a) At least 3 custom multi-table SQL queries supporting decision-making
- b) Interactive tables and visualisations via code, dashboard tool, or web app
- c) At least 2 data visualisations appropriate to analysis techniques
- Publicly accessible dashboard required
- Implementation process explained; viz choices justified with literature
